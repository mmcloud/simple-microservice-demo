# Frontend Bakery

Java servlets project serving the front end bakery

## Quickstart

1) Build Java binarys

```bash
./gradlew build
```

2) Run container
```
docker-compose up --build
```

3) Run in one command
```
./gradlew build & docker-compose up --build
```
## Useful commands
 - Stop container
```
docker-compose stop
```
- View container processors
```
docker-compose ps
```
